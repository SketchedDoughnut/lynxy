BUILD INSTRUCTIONS
- First, if not done so already, install build and twine and update. (python3 -m pip install (build / twine))
- Build. (python3 -m build)

RELEASE INSTRUCTIONS
- For test pypi: run: python3 -m twine upload --repository testpypi dist/*
- For normal pypi: python3 -m twine upload --repository pypi dist/*

DOWNLOAD INSTRUCTIONS
- For test pypi: pip install -i https://test.pypi.org/simple/ lynxy
- For normal pypi: pip install lynxy
